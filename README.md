# Financial Analytics Dashboard

An interactive financial performance dashboard built with Python, Dash, and Plotly.

## What it does
- Analyzes $174M in sales across 5 segments, 5 countries, and 3 years
- 8 fully interactive charts — all update in real time when filters change
- KPI cards: Total Sales, Total Profit, Profit Margin %, Units Sold
- Filters: Segment, Country, Year

## Charts included
- Sales vs Profit by Year (grouped bar)
- Profit Margin % Trend (line)
- Sales by Segment (bar)
- Profit by Country (horizontal bar)
- Top 6 Products by Profit (horizontal bar)
- Monthly Sales Trend (area)
- Discount vs Profit Margin (scatter bubble)

## How to run
pip install dash plotly pandas openpyxl numpy
python app.py

Then open http://127.0.0.1:8050
